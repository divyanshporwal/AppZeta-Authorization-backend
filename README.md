# AppZeta Authorization Backend

**Backend API for user authentication & authorization** using Node.js, Express, JWT and MongoDB/Mongoose.

This project provides a REST API for signing up users, logging in, and protecting routes using JSON Web Tokens (JWT). The backend code is designed to be simple, modular, and easy to extend for future use.
